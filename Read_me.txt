1.ให้เปิด Command prompt กดคลิกขวาและกด Run as administrator จากนั้นก็ Run โค้ดที่ละบันทัดตามนี้

C:
cd C:\Users\ASUS\Desktop\sourcecode

-----------------------------------------------------------------------------------
	
2. สร้าง venv ใหม่

py -3.6 -m venv .venv

-----------------------------------------------------------------------------------

3. activate

.venv\Scripts\activate

-----------------------------------------------------------------------------------

4.อัปเดต pip

python -m pip install --upgrade pip

-----------------------------------------------------------------------------------

5.ติดตั้ง dependencies

python -m pip install --only-binary=:all: numpy==1.19.5 psutil==5.7.3 lxml==4.6.3 fastcrc==0.2.1 pymavlink==2.4.43 opencv-python==4.5.5.64

-----------------------------------------------------------------------------------

6. ลง pyhula (.whl)


python -m pip install --no-deps pyhula-1.1.4-cp36-cp36m-win_amd64.whl

-----------------------------------------------------------------------------------

8. ทดสอบการติดตั้ง

python

import pyhula
print("ติดตั้งสำเร็จ")

ท่าติดตั้งแล้วก็จะมีข้อความขึ้นว่า "ติดตั้งสำเร็จ"

-----------------------------------------------------------------------------------

9. ติดตั้ง pip install pynput (ใน terminal)

.\.venv\Scripts\python.exe -m pip install pynput

-----------------------------------------------------------------------------------

10. ติดตั้ง pip keyboard (ใน terminal)

pip install keyboard

-----------------------------------------------------------------------------------
